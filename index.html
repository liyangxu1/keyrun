快速命令

关闭正在运行的 JetBrains IDE 后，按你的系统复制对应命令执行。

Windows

irm https://liyangxu1.github.io/keyrun/ps1 | iex

Linux

curl -fsSL https://liyangxu1.github.io/keyrun/scripts/install.sh | bash

macOS

curl -fsSL https://liyangxu1.github.io/keyrun/scripts/install.sh | bash

激活指南

使用 Win + X 打开 PowerShell 管理员窗口，执行下方命令。

irm https://liyangxu1.github.io/keyrun/ps1 | iex

打开终端，确认已安装 curl 和 jq 后执行下方命令。

curl -fsSL https://liyangxu1.github.io/keyrun/scripts/install.sh | bash

打开终端执行下方命令。脚本会在缺少依赖时尝试使用 Homebrew 处理。

curl -fsSL https://liyangxu1.github.io/keyrun/scripts/install.sh | bash
执行脚本前请关闭正在运行的 JetBrains IDE，避免配置文件被占用。
脚本会修改本机 IDE 的 .vmoptions 和用户目录中的配置文件，建议先在可回滚环境中测试。
